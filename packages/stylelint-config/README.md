# `@cqfe/stylelint-config`

> stylelint config used by `cqfe` team

## Usage

- `package.json`

```json
{
  "stylelint": {
    "extends": "@cqfe/stylelint-config"
  }
}
```

- `.stylelintrc`

```json
{
  "extends": "@cqfe/stylelint-config"
}
```

- `stylelint.config.js`

```js
module.exports = {
  extends: '@cqfe/stylelint-config',
}
```
